# 🧪 Backend Experiments

A collection of hands-on backend experiments focused on understanding and implementing core server-side concepts using Node.js.
This repository serves as a structured learning space for exploring real-world backend features in a simplified and practical way.

---

## 🎯 Purpose

This repository is built to:

- Reinforce backend fundamentals through small, focused experiments
- Explore real-world backend concepts without over-engineering
- Maintain a clean and organized record of learning progress
- Prepare for building production-level backend applications

---

## 🧠 Topics Covered

The experiments in this repository include:

- 🔐 Authentication (JWT, Sessions, Cookies)
- 🧾 Request Validation (Zod / custom validation)
- 🗄️ Database Operations (CRUD, Joins, Aggregations)
- 📄 Pagination & Filtering
- 🚦 Rate Limiting
- 🌐 CORS handling
- 🔑 Password Hashing (bcrypt)
- 🛡️ Middleware patterns
- 📦 API design fundamentals

---

Each folder represents an independent experiment focusing on a specific concept.

---

## ⚙️ How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/PratikKrGhosh/backend-experiments.git
   ```

2. Navigate to any experiment:

   ```bash
   cd backend-experiments/auth/jwt-auth
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Run the project:
   ```bash
   npm run dev
   ```

## 📌 Notes

- Each experiment is intentionally kept minimal and focused
- Code is simplified for learning purposes, not production use
- Some experiments may be incomplete as part of the learning process
